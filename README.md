# wirvsvirushackathon - PAUSEN MONSTER

## Architecture

### Frontend

ReactNative App

Build for

- Appstore (iOS)\*
- Playstore (Android)
- Web

Used Packages:

- React Native
- React Native Web
- React Navigation
- Styled Components
- Expo
- Expo web
- AWS Cognito for authentication
- AWS DataStore

### Backend

AWS DataStore
AWS Cognito

### AppSync

User generated content will be seamlessly synced to the cloud, and back to all client devices
